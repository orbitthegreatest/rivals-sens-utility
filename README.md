# Rivals Sens Utility

A desktop app for Rivals sensitivity conversion, DPI calculation, sens finder, profiles, and more.

## How to get your .exe

1. **Create a free GitHub account** at https://github.com if you don't have one
2. **Create a new repository** — click the `+` in the top right → "New repository"
   - Name it anything, e.g. `rivals-sens-utility`
   - Set it to **Public** or **Private** (both work)
   - Click **Create repository**
3. **Upload these files** — drag and drop all files from this folder into the repo page, then click **Commit changes**
4. **Go to the Actions tab** — GitHub will automatically start building
5. **Wait ~3 minutes** for the build to finish (green checkmark ✅)
6. **Download your .exe** — click the build, scroll down to **Artifacts**, and download either:
   - `RivalsSensUtility-Setup` → installs the app properly (recommended)
   - `RivalsSensUtility-Portable` → single `.exe`, no install needed

Every time you push new changes, it rebuilds automatically.

## Features

- 🎯 Accurate sensitivity conversion
- 🖱️ DPI calculator
- 🎮 Roblox sensitivity calculator
- 🔍 Sens finder with cm/360° feedback
- 🔄 Game crossover (Valorant, CS2, Fortnite, Overwatch 2, Apex, R6)
- 📊 Sensitivity comparison chart
- 🖱️ Mousepad size check
- ⭐ Save unlimited presets
- 👤 Mouse DPI profiles
- 💾 Auto-remembers your last setup
